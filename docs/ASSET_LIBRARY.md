## üìê IMPLEMENTATION GUIDE

### Adding Borders to Drug Cards:
1. Save border images as `/public/borders/{drug-id}-border.png`
2. Update DoseCard.tsx to use border images:
```tsx
<div 
  className="relative"
  style={{
    backgroundImage: `url(/borders/${dose.id}-border.png)`,
    backgroundSize: 'contain',
    backgroundRepeat: 'no-repeat',
    padding: '12px'
  }}
>
  {/* Card content */}
</div>
```

### Adding Icons:
1. Save icons as `/public/icons/{drug-id}-icon.png`
2. Replace emoji icons with image icons in DoseCard

### File Structure:
```
/public
  /borders
    psilocybin-border.png
    dmt-border.png
    ...
  /icons
    psilocybin-icon.png
    dmt-icon.png
    ...
  /backgrounds
    main-bg.jpg
    hero-bg.jpg
    ...
  /logo
    logo-full.svg
    logo-icon.svg
    logo-wordmark.svg
```
